## Java 上机实验 5

## 1. 接口编程

按以下要求编程：

### (1) 编写两个接口 `Shape` 和 `Knowledge`：
- `Shape` 定义两个方法 `area()` 和 `perimeter()`（分别返回该形状的面积和周长）；
- `Knowledge` 定义两个方法 `areaFormula()` 和 `perimeterFormula()`（分别返回该形状的面积计算公式和周长计算公式的语言描述）。

### (2) 编写两个类 `Rectangle` 和 `Triangle`（矩形和三角形）：
- 两个类均要实现上述两个接口，并编写相应的方法的实现代码。

### (3) 编写两个类 `Square` 和 `EquilateralTriangle`（正方形和等边三角形）：
- `Square` 继承 `Rectangle`，并重写上述 4 个方法；
- `EquilateralTriangle` 继承 `Triangle`，并重写上述 4 个方法。

## 2. 一维数组编程

现有两个 double 类型的数组如下：
```java
double xs[] = {1, 9, 3, 5, 6};
double ys[] = {10, 11, 4, 7, 13};
```
又有一个矩形的数组如下（实现采用上一题的）：
```java
Rectangle rects[] = new Rectangle[10];
```
要求：
1. 构造 5 个矩形的实例，并放入数组 `rects`（构造时候的长和宽分别为 `xs` 和 `ys` 对应位置的元素）；
2. 按面积由小到大对这 5 个矩形实例进行排序。
> 说明：不知道大家数据结构的排序有没有学，如果没有学第(2)条就忽略。

## 3. 二维数组编程

编写一个矩阵类 `MyMatrix`，要求：
1. 构造的时候用户可以指定矩阵的行数和列数；
2. 矩阵的每个元素随机生成，并采用二维数组保存数据；
3. 编写一个方法 `maxNum`，返回二维数组中最大的元素。
> 提示：生成随机数据的代码如下：
```java
import java.util.Random;
Random rand = new Random();
int randomNumber = rand.nextInt();
```

## **1. 判断回文（Palindrome）**

题目：  
判断用户输入的字符串是否为回文。回文指正反读都一样的字符串，例如 “radar” 是回文，“hello” 不是。

提示：  
- 读取用户输入的一行字符串。  
- 可在判断前去掉空格并忽略大小写，以支持更宽泛的回文判定（例如忽略空格和大小写后判断）。

示例：  
- 输入：Radar  
- 说明：忽略大小写后为 “radar”，正反一致。  
- 预期输出：该字符串是回文（或等价描述）。

备注：  
如需更严格判断，可不去除空格或不忽略大小写，具体根据题目要求调整。

## **2. 对数字形式的字符串排序（从小到大）**

题目：  
给定字符串集合 {"20", "50", "40", "30", "60", "55", "80"}，要求根据数字大小从小到大排序并输出结果（仍以字符串形式表示）。

提示：  
- 将每个字符串转换为整数进行比较或排序。  
- 排序完成后再将结果转换为字符串输出，保证输出格式与输入形式一致。

示例输入：  
{"20", "50", "40", "30", "60", "55", "80"}

示例输出（按照数字从小到大）：  
["20", "30", "40", "50", "55", "60", "80"]

备注：  
若集合中包含非数值字符串，请先进行验证或处理错误情况（例如跳过或报错）。

## **3. 统计段落中的单词信息**

题目：  
用户输入一段带空格的文本（代表一段话），要求统计下列信息：  
1) 单词总数；  
2) 单词 "and" 出现的次数（忽略大小写）；  
3) 包含子串 "ing" 的单词个数（忽略大小写）。

实现要点与提示：  
- 以空白字符分割文本为基本 token（词元）。  
- 去除每个 token 两端的标点符号（如句号、逗号、分号、引号等）再作为单词处理，以避免计入标点。  
- 统计时均使用忽略大小写的比较方式，使 "And"、"AND" 等均计为 "and"。  
- 包含 "ing" 的判断为子串包含关系，例如 "programming"、"computing"、"beginning" 都应计数。

示例文本（用于验证）：  
Java is a programming language and computing platform first released by Sun Microsystems. It has evolved from humble beginnings to power a large share of digital world by providing the reliable platform upon which many services and applications are built.

示例预期统计结果：  
- 单词总数: 40  
- "and" 出现次数: 2  
- 包含 "ing" 的单词数: 4

备注：  
不同的分词或标点处理策略可能导致单词计数在极端情况下有少量差异，常见做法为上述去标点再计数以获得稳定结果。
